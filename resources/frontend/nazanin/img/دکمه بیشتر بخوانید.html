در اینجا یک کد ساده React (با Tailwind CSS برای استایل) برای دکمه «بیشتر بخوانید» ارائه می‌دم که در ابتدا فقط بخشی از متن رو نشون می‌ده و با کلیک روی دکمه، متن کامل نمایش داده می‌شه:

jsx
import React, { useState } from 'react';

export default function ReadMore() {
  const [showFullText, setShowFullText] = useState(false);

  const text = `لورم ایپسوم متن ساختگی با تولید سادگی نامفهوم از صنعت چاپ و با استفاده از طراحان گرافیک است. چاپگرها و متون بلکه روزنامه و مجله در ستون و سطر آنچنان که لازم است...`;

  const shortText = text.slice(0, 80) + '...';

  return (
    <div className="max-w-md mx-auto p-4">
      <p className="text-gray-800 leading-relaxed">
        {showFullText ? text : shortText}
      </p>
      <button
        onClick={() => setShowFullText(!showFullText)}
        className="mt-2 text-blue-500 hover:underline focus:outline-none"
      >
        {showFullText ? 'کمتر بخوانید' : 'بیشتر بخوانید'}
      </button>
    </div>
  );
}

توضیح:

  • showFullText: مشخص می‌کنه متن کامل نشون داده بشه یا خیر.

  • ابتدا فقط ۸۰ کاراکتر اول متن نمایش داده می‌شه.

  • با کلیک روی دکمه، setShowFullText تغییر می‌کنه و متن کامل (یا خلاصه) نمایش داده می‌شه.

اگر با React کار نمی‌کنی و به نسخه ساده HTML + JavaScript نیاز داری، بگو تا اونم برات بنویسم.

می‌گه:
«اگر عنصر .more کلاس show نداشته باشه، اضافه‌اش کن؛ اگر داشته باشه، حذفش کن.»

و در CSS ما نوشتیم که:

css
.more {
  display: none;
}

  display: block;
}

پس اضافه شدن کلاس show باعث نمایش متن مخفی می‌شه.

javascript
btn.textContent = moreText.classList.contains('show') ? 'کمتر بخوانید' : 'بیشتر بخوانید';

✅ این یک شرط خلاصه است (ternary operator):

  • اگر کلاس show وجود داشته باشه → متن دکمه می‌شه "کمتر بخوانید"

  • اگر نداشته باشه → می‌شه "بیشتر بخوانید"
